Open the terminal application and then start typing these commands to know your Linux desktop or cloud server/VM

Get free and used memory

Are you running out of memory?Use the free command to show the total amount of free and used physical(RAM) and swap memory in the Linux system.It also displays the buffers and caches used by the kernel.We must use the dmidecode command if we want information about:

  • memory configuration
  • maximum supported memory by the Linux server
  • Linux memory speed

 
            free 
            #human readable outputs
            free -h 
            #use the cat command to find geeky details
            cat /proc/meminfo


            sudo dmidecode -t memory

Probe for hardware

We can quickly probe for the hardware present in the Linux server or desktop.Alternatively,you may find the lshw command useful to display your Linux hardware information 

 
            #find detailed info about the Linux box
            hwinfo 
            #show only a summary# 
            hwinfo --short 
            #view all disks# 
            hwinfo --disk 
            #get an overview# 
            hwinfo --short --block
            #find a particular disk# 
            hwinfo --disk --only /dev/sda 
            hwinfo --disk --only /dev/sda 
            #try 4 graphic cards ports for monitor data#
            hwprobe=bios.ddc.ports=4 hwinfo --monitor 
            #limit info to specific devices# 
            hwinfo --short --cpu --disk --listmd --gfxcard --wlan --printer 

 
            
            sudo lshw -short 
            inxi -Fxz

Know yourself

Display Linux user and group information for the given user name.If user name omitted show information for the current user by using id command. You can also see who is logged on your Linux server using who and who am i 

 
            id 


            who  
            who am i

List block storage devices

All Linux block devices give buffered access to hardware devices and allow reading and writing blocks as per configuration.Linux block devices has names.You can list them easily using a specific syntax. 

 
        lsblk  
        #list only# 
        lsblk -l 
        #filter out loop devices using the grep command# 
        lsblk -l | grep '^loop'

Linux distribution information

The command lsb_release is useful if we want to get distribution-specific information such as,description of of the currently installed distribution,release number and code name.No LSB modules are available.


        lsb_release -a

Display info about the CPUs

The lscpu command gathers and displays CPU architecture information in an easy-to-read format for humans including various CPU bugs.CPU can be listed using the lshw command too.


        lscpu 
                
        sudo lshw -C cpu

Display hardware topology

If you want to see the topology of the Linux server or desktop try lstopo and lstopo-no-graphics.You will see information about:

  • Numa memory nodes
  • shared caches
  • CPU packages
  • processor cores
  • thread and more

            
        lstop
        lstopo-no-graphics

List usb devices

We all use USB devices,such as external hard drives and keyboards.Run the lsusb command for displaying information about USB buses in the Linux system and the devices connected to them.We use the lspci command for displaying information about PCI buses in the system and devices connected to them.

            
        lsusb
        #want a graphical summmary of USB devices connected to the system# 
        sudo usbview 


        lspci

View current date and time zone

Typically we use the date command to set or get date/time information on the CLI. However,modern Linux distro use the timedatectl command to query and change the system clock and its settings,and enable or disable time synchronization services.


        date 


        timedatectl

Who is logged in

Run the w command on Linux to see information about the Linux users currently on the machine,and their processes.


        $ w
